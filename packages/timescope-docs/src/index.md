## Key Features

- **Infinite Time Range**: Supports panning and zooming across timelines without fixed boundaries, using arbitrary-precision decimal arithmetic
- **Chart Composition**: Constructs charts by combining marks and links to create envelopes, scatter plots, impulse charts, and other visualizations
- **Chunked Data Loading**: Loads and unloads data chunks dynamically based on viewport position and zoom level
- **TypeScript Support**: Includes type definitions for configuration options, event payloads, and API methods
